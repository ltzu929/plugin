## 🎯 核心功能

### 1. 📈 弹幕密度分析
- **1分钟间隔统计**: 精确展示每个时间段的弹幕数量
- **高峰时段识别**: 自动标记弹幕密度最高的精彩时刻
- **多维度展示**: 分别统计总数、中文、英文、表情弹幕
- **趋势分析**: 移动平均算法平滑数据曲线

### 2. 🎯 热门弹幕实时悬浮窗
- **鼠标跟随设计**: 热门弹幕列表显示在鼠标旁边
- **±30秒时间窗口**: 智能分析当前时间点前后30秒内的弹幕
- **Top 10排行榜**: 自动统计出现频率最高的前10个弹幕内容
- **实时更新**: 鼠标移动时即时更新热门弹幕列表

### 3. 🔍 智能数据筛选
- **语言筛选**: 支持只看中文、英文或带表情的弹幕
- **高峰筛选**: 可以只显示弹幕密度高的时间段
- **关键词搜索**: 快速定位特定关键词在弹幕中的出现情况
- **数据导出**: 支持将分析结果导出为JSON格式

### 核心算法
```typescript
// 热门弹幕识别算法（±30秒时间窗口）
const getPopularDanmaku = (danmakus: DanmakuItem[], currentTime: number) => {
  const timeWindow = 30; // 30秒窗口
  const nearbyDanmakus = danmakus.filter(d => 
    Math.abs(d.time - currentTime) <= timeWindow
  );
  
  // 统计相同内容出现次数
  const contentCount = countDuplicateContent(nearbyDanmakus);
  
  // 返回Top 10热门弹幕
  return Object.entries(contentCount)
    .map(([content, count]) => ({ content, count }))
    .sort((a, b) => b.count - a.count)
    .slice(0, 10);
};
```
